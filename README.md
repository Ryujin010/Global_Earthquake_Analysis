# 🌍 Global Earthquake Analysis

## 📊 Descripción
Proyecto de análisis de datos sísmicos históricos (1900–2026), enfocado en identificar patrones temporales, geográficos y de magnitud.

## 🚀 Funcionalidades
- Limpieza y validación de datos
- Análisis exploratorio (EDA)
- Visualizaciones avanzadas
- Dashboard interactivo con Streamlit

## 🧠 Insights
- Alta concentración en zonas tectónicas
- Baja correlación entre profundidad y magnitud
- Incremento en registros a lo largo del tiempo

## 🛠️ Tecnologías
- Python
- Pandas
- Matplotlib
- Seaborn
- Streamlit
